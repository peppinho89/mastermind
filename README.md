mastermind
==========

The Mastermind game in C - Multiplayer

Author: Giuseppe Pace - Cosimo Bellini

==========
Features:

- Settings:
	
	- Game Mode: Human vs Human - Human vs CPU
	- Nickname of players
	- Length of Secret Code
	- Number of different code elements
	- Number of shot

	
- Save the match and resume it.

- Top Ten Chart
